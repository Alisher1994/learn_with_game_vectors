# Вектор-баттл

Интерактивная игра-презентация по теме «Векторы» для 7–8 класса: две команды (синие и красные), вопросы на большом экране, ответы с двух телефонов по QR-коду, учёт времени и очков, таблица рейтингов.

## Локальный запуск

```bash
npm install
npm run dev
```

- Фронтенд: http://127.0.0.1:5173  
- API и WebSocket: http://127.0.0.1:3001  

Откройте главную страницу, нажмите «Новая игра» — откроется экран учителя с двумя QR-кодами. С телефонов перейдите по ссылкам или отсканируйте коды, выберите команду на экране присоединения.

## Деплой (Railway + GitHub)

1. Залейте репозиторий на GitHub.
2. В Railway: New Project → Deploy from GitHub, выберите репозиторий.
3. Укажите **Root Directory** при необходимости (корень проекта).
4. Railway подставит переменную `PORT` — приложение уже читает `process.env.PORT`.
5. Сборка: `npm run build`, старт: `npm start`.
6. Данные рейтинга и справочника классов хранятся в файле `data/store.json` на диске. Для постоянства между перезапусками добавьте **Volume** в Railway и смонтируйте её в `/app/data`.

## Справочник классов

Раздел «Справочник классов и ФИО» на главной — учитель может заранее внести классы и списки учеников; на телефонах при выборе класса поля заполняются быстрее.

## Правила очков

- За вопрос: базово **10** очков за верный ответ.
- Если верны обе команды: быстрейшей дополнительно **+5** (итого 15 и 10).
- Если верна одна команда: ей **15** очков за этот вопрос.
